liblzma-jna
===========

JNA-based Java bindings to liblzma.

Status
------
A work-in-progress; Check back later.

Requirements
------------
* [liblzma](http://tukaani.org/xz) (`apt-get install liblzma5`) 
* [Maven](https://maven.apache.org/) (`apt-get install maven`)

Build
-----
    $ mvn package

Tests
-----
    $ mvn test
